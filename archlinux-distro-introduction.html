Arch Linux distro - Introduction


Arch Linux is a simple, lighweight Linux based distribution.
https://www.archlinux.org/


It follows a rolling-release model (current version is always up to date)

It provides a very comprehensive documentation within its wiki.


Arch Linux overview:
https://wiki.archlinux.org/index.php/Arch_Linux


Arch compared to other distributions:
https://wiki.archlinux.org/index.php/Arch_compared_to_other_distributions



Package Manager and Build System


pacman: a lightweight, simple and fast package manager
https://wiki.archlinux.org/index.php/Pacman

Arch Build System, a ports-like system to make it easy to build and install packages from source.
https://wiki.archlinux.org/index.php/Arch_Build_System


The goal of pacman is to make it possible to easily manage packages, whether they are from the official repositories or the user's own builds.
